About ViewSpeedFlow

ViewSpeedFlow is a real-time network monitoring tool for Windows, designed to track your download and upload speeds, run speed tests, and display system specifications in a simple and efficient way. This is an early access version, so expect UI improvements and new features in upcoming updates!

×

Current Features:

  • Real-Time Network Speed Monitoring – View current download and upload speeds in the taskbar tooltip.
  • Speed Test Functionality – Run a speed test to measure your actual download and upload speeds with a graphical representation of the results.
  • Network Topology Detection – Displays whether you're using Ethernet or Wi-Fi for a better understanding of your connection.
  • System Specifications Overview – Get detailed system specs, including processor details, RAM, disk space, Python version, and OS info.
  • Crucial Network Details – View active connections, public IP, hostname, and OS (with an option to hide sensitive details).
  • Light & Dark Mode Support – Switch between dark mode for a sleek look and light mode for better visibility.
  • Export Results – Save your speed test results as a PDF for easy reference.

Upcoming Improvements:

  • (soon) Complete UI Overhaul – A more modern and polished look is in development.
  • (soon) New Features & Enhancements – Expect better customization options, additional network diagnostics, and more...
